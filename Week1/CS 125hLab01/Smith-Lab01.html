What is the difference between the World Wide Web and the Internet?

The Internet is a group of interconnected Wide Area Networks (WANs) connected by Internet Exchange Points (IXPs). The World Wide Web (WWW) is a part of the Internet that allows users to access information through a variety of protocols such as Dynamic Naming System (DNS) that translates Internet Protocol (IP) addresses to Uniform Resource Locators (URLs), the HyperText Transfer Protocol (HTTP) that requests webpage information from servers, and Transport Layer Security (TLS) that creates a secure HTTP connection (HTTPS). These WWW protocols are used on top of Internet protocols such as Transmission Control Protocol (TPC), Internet Protocol (IP) (which are often used together), as well as the User Datagram Protocol (UDP).

How is the server role different when processing a static web page versus a dynamic web page?

When processing a static web page, the server send the page directly to the client without any modifications. In contrast, the content of a dynamic web page is populated by a script or application on the web server in response to information sent via the HTTP request.

How do web developers know which HTML tags will work in their code?

HTML is a standardized language and a developer can refer to a resource such as the World Wide Web Consortium (W3C) to ensure the tags they are trying to use are valid.

Why is it important to create web pages in such a way so that they are accessible in the widest possible array of browsing devices, including non-visual devices?

Web sites should be coded and designed in a way that anyone can access them. The aesthetic component is a core part of responsive design where webpages scale or have different layouts based on the browsing device. Using good HTML coding practices, such as using < em > < /em > rather than < b > < /b > so that screen readers will know to emphasize the text.

Universal design, or designing something in a way that it can be accessed by the largest audience possible, is important so that everyone can access the information. Some examples of this are writing webpage code in a way that screen readers can read the text, using alt-text for images so that screen readers can describe to the user what an image is, using appropriate color combinations for visually impaired users, and making websites that are usable without a mouse.
